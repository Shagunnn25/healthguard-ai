# HealthGuard AI

AI-powered medical assistant system — fully self-hosted, no external APIs.

## Modules
- Disease Predictor (symptoms → conditions)
- Lab Report Analyzer
- Medical QA Engine (RAG-based)
- OCR Module
- Summarization Engine
- Misinformation Detector
